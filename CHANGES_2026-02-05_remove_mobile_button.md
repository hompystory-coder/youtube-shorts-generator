# 🗑️ 모바일 미리보기 버튼 제거

## 변경 내용

모바일에서 나타나는 플로팅 미리보기 버튼과 오버레이를 제거했습니다.

### 제거된 항목

1. **플로팅 버튼** (1943-1952번 라인)
   ```jsx
   <div className="lg:hidden fixed bottom-6 right-6 z-50">
     <button>📱 미리보기</button>
   </div>
   ```

2. **모바일 오버레이** (1955-2049번 라인)
   - 전체 화면 오버레이
   - 9:16 미리보기
   - 닫기 버튼

3. **State** (67번 라인)
   ```jsx
   const [showMobilePreview, setShowMobilePreview] = useState(false);
   ```

## 이유

데스크톱 레이아웃이 완성되어 오른쪽에 항상 미리보기가 표시되므로, 모바일 전용 버튼이 불필요합니다.

## Before

```
┌──────────────────────────┐
│                          │
│  [설정 패널]             │
│                          │
│                    [📱]  │ ← 우하단 플로팅 버튼
└──────────────────────────┘
```

클릭 시 → 전체화면 오버레이 표시

## After

```
┌──────────────────────────────────────┐
│  [설정 700px]  │  [미리보기 380px]   │
│                │                      │
│                │  (항상 표시)         │
└──────────────────────────────────────┘
```

플로팅 버튼 없음 - 미리보기가 이미 화면에 있음
