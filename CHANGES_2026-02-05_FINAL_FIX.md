## 진짜 진짜 근본 원인

**1787번 라인에 `</div>`가 하나 더 있어서, flex-row가 조기에 닫혔습니다!**

### 문제 구조
```jsx
<div className="flex flex-row gap-8">
  <div className="w-[700px]">
    <!-- 설정 패널 -->
  </div>  
  </div>  // ← 여기서 설정 패널 닫기
</div>    // ← 여기서 flex-row가 닫힘 (잘못됨!)

{/* 미리보기가 밖으로 나감! */}
<div className="w-[380px]">
  <!-- 미리보기 -->
</div>
```

### 해결
```jsx
<div className="flex flex-row gap-8">
  <div className="w-[700px]">
    <!-- 설정 패널 -->
  </div>  // ← 설정 패널 닫기 (여기만!)
  
  {/* 미리보기가 안에 들어옴! */}
  <div className="w-[380px]">
    <!-- 미리보기 -->
  </div>
</div>  // ← 여기서 flex-row 닫기
```
